# My News
It's a Android Application for get the latest news from The New York Times. You can choice between 3 categories :
- TopStories
- MostPopular
- Sports

<img src="./app/src/main/res/drawable/mynews.png?Raw=true" width="30%" height="30%">

You can search for a special news; All the result will be display in a new screen, it's a list of all articles find and you can click on for see the detail.

<img src="./app/src/main/res/drawable/myNews.png?Raw=true" width="30%" height="30%">

You can see the detail of a new by clicking on the article.

<img src="./app/src/main/res/drawable/myNews.png?Raw=true" width="30%" height="30%">

You can activate a notification for a special query or for your preferences and the notification send once per day.

<img src="./app/src/main/res/drawable/webview.jpg?Raw=true" width="30%" height="30%">
